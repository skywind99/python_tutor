# 2026-05-16 작업 기록

## 버그 수정
- **문제관리 학생 풀이 미표시** — 교사가 반을 2개 이상 가질 경우 `.single()` 쿼리 오류 발생. 배열 쿼리 + `.in('class_id', classIds)` 방식으로 수정
- **AI 모범풀이 미작동** — 모델명 `gemini-1.5-flash` → `gemini-2.0-flash` 변경, 에러 로그 추가
- **튜토리얼 퀘스트 1-3 답안 노출** — codeTemplate에서 `name = input(...)`, `song = input(...)` 라인 제거 → 주석으로 대체
- **튜토리얼 퀘스트 4-3 답안 노출** — for 변수명이 보이는 구조 수정, multiblank 방식으로 변경
- **튜토리얼 페이지 상태 미초기화** — 퀘스트 이동 시 이전 입력값이 남아 있던 문제. `key={page.id}` 추가로 컴포넌트 강제 리마운트

## 신규 기능
- **로그인 방식 변경** — 이메일 전용 → 아이디/이메일 듀얼 지원 (`@` 포함 시 이메일, 미포함 시 `아이디@pytutor.local` 변환)
- **페이지 너비 통일** — 학습자료·랭킹·내정보·학생관리 페이지를 `max-w-6xl`로 통일 (문제관리 기준)
- **학습자료 헤더 개선** — 다른 페이지와 동일한 스타일(아이콘·부제목)로 통일
- **교사 모범풀이 버튼** — 미션 페이지 실행결과 영역에 교사 전용 🔍 모범풀이 버튼 추가
- **AI Groq 폴백** — AI 힌트·모범풀이에서 Gemini 실패 시 Groq(`llama-3.3-70b-versatile`)로 자동 전환
- **튜토리얼 DB 연동** — 완료 페이지마다 `xp_logs`에 저장 (`source_id: tutorial-{missionId}-{pageId}`)
- **학습자료 튜토리얼 카드** — 학습자료 최상단에 튜토리얼 진행률 카드 추가 (보라색 그라디언트, 프로그레스 바)
- **추가문제 수정 기능** — 교사가 AI 생성 문제를 수정 가능(제목·주제·설명·예상출력·난이도)
- **추가문제 완료자 표시** — 추가문제 카드에 완료 학생 이름 목록 표시
